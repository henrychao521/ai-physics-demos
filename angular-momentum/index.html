力學・轉動與角動量

角動量守恆（溜冰者效應）

沒有外力矩時角動量 L = Iω 守恆。把配重往內收，轉動慣量 I = I₀ + 2mr² 變小，角速度 ω = L/I 就變大—— 溜冰者收手轉得更快。注意此時轉動動能 KE = ½Iω² 反而增加，多出的能量來自你「收手」做的功。

情境：
物理重點：角動量 L = Iω，轉動慣量 I = I₀ + 2mr²（配重離軸越遠 I 越大）。系統不受外力矩時 L 守恆， 所以收手（r↓）→ I↓ → ω = L/I ↑，轉得更快。但 轉動動能 KE = ½Iω² = L²/(2I)：I 變小時 KE 反而變大， 這些能量是你克服離心傾向把配重拉向中心所做的功（不是能量憑空出現）。星球塌縮加速自轉、跳水team收身翻轉，都是同一原理。
