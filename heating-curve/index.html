熱學・比熱與相變

比熱與加熱曲線

用固定功率加熱 −20°C 的冰。溫度沿著曲線上升，但在熔化（0°C）與沸騰（100°C）時出現 水平平台——加熱卻不升溫，因為熱量都用來改變相態（潛熱）。斜段則服從 Q = mcΔT。

情境：
溫度 — 加熱時間曲線（平台＝相變吸潛熱）
物理重點：斜段（升溫）吸收的是顯熱 Q = m·c·ΔT，比熱 c 越大、質量 m 越大，同樣加熱升溫越慢（斜率越緩）。 平台（相變）吸收的是潛熱 Q = m·L：熔化吸熔化熱 L_f、沸騰吸汽化熱 L_v，期間分子忙著掙脫彼此的束縛， 動能（溫度）不變。水的汽化熱（2260 J/g）遠大於熔化熱（334 J/g），所以沸騰平台特別長——這也是流汗、蒸氣燙傷的物理。
