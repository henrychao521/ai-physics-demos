電磁波（E⊥B 同相傳播）

變化的電場產生磁場、變化的磁場又產生電場——彼此「接力」向前，就是電磁波。電場 E（紅）與磁場 B（藍）互相垂直、 也都垂直於傳播方向，且同相振盪。不論頻率高低，真空中速度恆為光速 c = λf。光，就是一段可見的電磁波。

波段：
物理重點：馬克士威方程組預測電磁波，並算出其速度恰為光速 c = 1/√(ε₀μ₀) ≈ 3×10⁸ m/s，證明光是電磁波。 真空中所有電磁波都以 c 行進：c = λf，所以頻率越高、波長越短。電場與磁場振幅比固定：E₀ = c·B₀。 把電磁波看成光子時，能量 E = h·f（頻率越高、光子能量越大，這就是 X 射線、γ 射線具穿透/游離能力的原因）。 電磁波譜：無線電 → 微波 → 紅外 → 可見光（380–750 nm） → 紫外 → X 射線 → γ 射線，本質相同、只差頻率。
