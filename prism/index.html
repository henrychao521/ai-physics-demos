光學・色散

稜鏡色散（白光分成彩虹）

白光其實是各色光的混合。玻璃的折射率會隨波長改變——波長越短（越偏紫）折射率越大、偏折越多。 所以白光通過三角稜鏡兩次折射後，各色被拉開成一道光譜：紅光偏折最少、紫光最多。這就是牛頓的稜鏡實驗，也是彩虹的成因。

情境：
物理重點：同一介質對不同波長的光有不同折射率，稱為色散。常用柯西公式 n(λ) = A + B/λ²： λ 越短 n 越大。每個面都遵守司乃耳定律 n₁ sinθ₁ = n₂ sinθ₂，稜鏡讓光折射兩次，總偏向角 δ = θ₁ + θ₂′ − A（A 為頂角）。因為紫光 n 最大，δ 也最大，故偏折最多、排在彩虹內側。 白光 → 紅橙黃綠藍靛紫，本質是「同一束光、不同波長走了不同角度」。雨後彩虹則是陽光在水滴中折射＋反射的色散。
