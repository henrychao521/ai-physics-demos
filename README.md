# 物理互動模擬平台（ai-physics-demos）

涵蓋**國高中物理**六大領域的瀏覽器互動模擬，共 **57 個模擬**。每個都可即時調參數、看公式與圖表變化，並附「物理重點」說明與**即時代入計算公式**面板。
純前端 HTML / Canvas，無外部相依、無建置步驟，適合課堂教學、課後自學與成果展示。

> 所有模擬皆經**物理數值驗證**：守恆律（能量、動量、角動量）、公式關係（克卜勒三定律、PV=nRT、法拉第定律、Snell 定律、光電效應 KE=hf−W、卡諾效率 η=1−Tc/Th、維因位移 λ_max=b/T、馬呂士 I=I₀cos²θ、LC 振盪 f=1/2π√(LC)⋯）以程式實測，與理論吻合。

> 📖 **開發歷程**：新模擬的逐段落開發過程與關鍵對話，見 [`docs/開發歷程.md`](docs/開發歷程.md)。

## 內容（依分館）

### 🟦 力學館（14）
- `kinematics-1d/` — 一維等加速度運動：v=u+at、x=ut+½at²，x-t／v-t／a-t 三圖連動。
- `projectile/` — 拋體運動：初速/仰角/重力/空氣阻力，拋物線與速度分量、v-t/y-t 圖。
- `energy-coaster/` — 能量守恆雲霄飛車：動能/位能/熱能三色長條，摩擦轉熱。
- `collision/` — 碰撞實驗台：彈性↔非彈性，動量守恆 vs 動能階梯損耗。
- `newton-force/` — 牛頓第二定律推箱子：自由體圖、合力、F=ma 與靜摩擦門檻。
- `incline/` — 斜面與力的分解：mg·sinθ / mg·cosθ，滑動門檻 tanθ vs μ，a=g(sinθ−μcosθ)。
- `circular/` — 等速圓周運動：向心加速度 a=v²/r，v⊥a，放手沿切線飛出。
- `angular-momentum/` — 角動量守恆（溜冰者效應）：L=Iω，收手 I↓→ω↑、KE=L²/2I。
- `shm/` — 簡諧運動三表徵：圓周投影＝彈簧振子＋單擺，正弦曲線與週期。
- `orbit/` — 行星軌道沙盒：萬有引力，克卜勒一/二/三定律、等面積。
- `vectors/` — 力的合成：可拖曳力向量，平行四邊形/頭尾相接，力平衡。
- `machines/` — 槓桿與滑輪：力矩平衡與省力倍數，功守恆。
- `buoyancy/` — 浮力與密度：阿基米德原理，沒入比例=ρ物/ρ液。
- `pascal-hydraulics/` — 帕斯卡原理/液壓機：P=F₁/A₁，F₂=F₁·A₂/A₁，省力不省功。

### 🟧 熱學館（5）
- `kinetic/` — 分子運動箱：氣體動力論，實測壓力 vs 理論 NkT/V，微觀導出 PV=nRT。
- `heating-curve/` — 比熱與加熱曲線：冰→水→蒸氣，潛熱平台與 Q=mcΔT。
- `gas-law/` — 理想氣體 PV 圖：P=nRT/V，等溫(波以耳)/等壓/等容(查理)過程。
- `heat-transfer/` — 熱傳導與熱平衡：熱由高溫流向低溫，終溫＝mc 加權平均（Q=mcΔT）。
- `heat-engine/` — 熱機・卡諾循環：Qh→W→Qc，PV 圖面積＝淨功，效率 η=1−Tc/Th（熱力學第二定律）。

### 🟩 波動館（6）
- `traveling-wave/` — 行進波（橫波）：y=A·sin(kx−ωt)，波速 v=fλ，質點只上下振動。
- `standing-wave/` — 駐波弦：共振、諧波 n=1,2,3、節點/腹點，f_n=nv/2L。
- `resonance-tube/` — 空氣柱共鳴管：開口端腹點/封閉端節點，開管 f=nv/2L、閉管 f=(2n−1)v/4L。
- `doppler/` — 都卜勒效應與音爆：波前壓縮/拉伸、頻率升降、超音速馬赫錐。
- `sound/` — 聲波合成器：響度/音調/音色，拍音（Web Audio 真實發聲）。
- `resonance/` — 受迫振盪與共振：A(f) 共振曲線與阻尼，共振橋/選台/樂器原理。

### 🟪 光學館（6）
- `lens/` — 透鏡成像（凸/凹）：三主光線追跡，1/f=1/p+1/q，實/虛、正/倒、放/縮。
- `mirror/` — 球面鏡成像（凹/凸）：f=R/2、1/f=1/p+1/q，刮鬍鏡與後照鏡原理。
- `refraction/` — 折射與全反射：Snell 定律、臨界角、光纖原理。
- `prism/` — 稜鏡色散：折射率 n(λ)，白光兩次折射分成彩虹，紫光偏折最多（牛頓稜鏡實驗）。
- `polarization/` — 偏振與馬呂士定律：I=I₀cos²θ，正交全暗，三片悖論；偏光眼鏡/LCD 原理。
- `color-mix/` — 加法混色盤：RGB 三光疊加（黃/青/洋紅/白），對照顏料減法。

### 🟥 電磁館（11）
- `circuits/` — 電路串並聯：歐姆定律 V=IR，串/並聯電流電壓分配與燈泡亮度。
- `efield/` — 電場與電位：點電荷電場向量、等位面、庫侖定律（可拖曳）。
- `capacitor/` — 平行板電容器：C=ε₀A/d、Q=CV、E=V/d、U=½CV²，介電質。
- `motor/` — 直流馬達：F=BIL 成力偶，τ=NBIA·sinθ，換向器維持同向旋轉（電→動）。
- `emwave/` — 電磁波：E⊥B 同相前進 c=λf，對數頻譜滑桿（無線電→γ）與光子能量 E=hf。
- `rc-circuit/` — RC 充放電：指數曲線、時間常數 τ=RC、63%/37%。
- `lc-oscillation/` — LC 振盪電路：電場能↔磁場能交換，f=1/2π√(LC)，與彈簧 SHM 類比。
- `current-magnetism/` — 電流的磁效應：直線/圓線圈/螺線管磁場與安培右手定則，B=μ₀I/2πr 等。
- `lorentz/` — 磁場中的帶電粒子：勞侖茲力 F=qv×B、迴旋半徑 r=mv/qB（質譜儀、極光）。
- `induction/` — 電磁感應發電機：法拉第定律 EMF=−N·dΦ/dt，磁通與電動勢 90° 相位。
- `transformer/` — 變壓器：Vs/Vp=Ns/Np，升壓必降流、功率守恆 VpIp=VsIs（高壓輸電）。

### ⬛ 近代物理館（8）
- `photoelectric/` — 光電效應：臨界頻率、強度無關、KE=hf−W，量子物理開端。
- `blackbody/` — 黑體輻射：維因 λ_max=b/T 峰移、史特凡 P=σT⁴，燈泡/太陽/天狼星顏色。
- `time-dilation/` — 時間膨脹（光鐘）：Δt=γΔt₀、γ=1/√(1−β²)，長度收縮 L=L₀/γ（狹義相對論）。
- `half-life/` — 放射性半衰期：N=N₀(½)^(t/T)、統計漲落，碳-14 定年。
- `nuclear-fission/` — 核分裂連鎖反應：增殖係數 k 控制臨界，每次約 200 MeV（E=Δmc²）。
- `bohr/` — 波耳原子模型：能階躍遷、氫原子光譜（Hα 656nm、巴耳末/來曼系）。
- `de-broglie/` — 物質波（德布羅意）：λ=h/p，波粒二象性與電子顯微鏡原理。
- `mass-energy/` — 質能等效與結合能：E=Δmc²，結合能曲線鐵-56 最高，分裂與融合都釋能。

### 🔬 波動光學・幾何光學（既有作品 / 精選）（5）
- `superposition/` — 干涉與摩爾紋：同一種疊加（一維拍頻，概念整合 hub，連到細部模擬）。
- `spectral-lab/` — 光譜干涉實驗室（歐幾里得視角）：3D 三原色建設性干涉，可調頻率/晶格夾角/色散（Plotly）。
- `diffraction/` — 繞射成像：雙/單狹縫干涉繞射，相位差與前後距離差 ΔL。
- `moire/` — 摩爾紋（Moiré）：兩組光柵疊加，微小差異被放大。
- `shadowless-lamp/` — 外科無影燈（2D 概念版）：多光源消影示意；3D 專業版見 [shadowless-lamp-sim](https://henrychao521.github.io/shadowless-lamp-sim/)（Three.js + IES 光錐）。

### 🛠 工程・資料（2）
- `hovercraft/` — RC 氣墊船工程模擬器：升力/推力馬達、電池、推重比與導風向量轉向（React + Babel）。
- `housing/` — 全台房市實證數據系統：縣市/行政區、季價量軌跡互動儀表板（Chart.js）。

> 註：`hovercraft/`、`housing/`、`spectral-lab/` 透過 CDN 載入函式庫，需連網；力學/熱學/波動/光學/電磁/近代各館為純 HTML/Canvas，可離線。
