# 物理互動模擬平台（ai-physics-demos）

涵蓋**國高中物理**六大領域的瀏覽器互動模擬。每個模擬都可即時調參數、看公式與圖表變化，並附「物理重點」說明。
純前端 HTML / Canvas，無外部相依、無建置步驟，適合課堂教學、課後自學與成果展示。

> 所有模擬皆經**物理數值驗證**：守恆律（能量、動量、角動量）、公式關係（克卜勒三定律、PV=nRT、法拉第定律、Snell 定律、光電效應 KE=hf−W、氫原子光譜⋯）以程式實測，與理論吻合。

> 📖 **開發歷程**：新模擬的逐段落開發過程與關鍵對話，見 [`docs/開發歷程.md`](docs/開發歷程.md)。

## 內容（依分館）

### 🟦 力學館
- `projectile/` — 拋體運動：初速/仰角/重力/空氣阻力，拋物線軌跡與速度分量、v-t/y-t 圖。
- `energy-coaster/` — 能量守恆雲霄飛車：動能/位能/熱能三色長條，摩擦轉熱（能量法積分，守恆 err=0）。
- `collision/` — 碰撞實驗台：彈性↔非彈性，總動量守恆 vs 動能階梯損耗（開放軌道，動量嚴格守恆）。
- `newton-force/` — 牛頓第二定律推箱子：自由體圖、合力、F=ma 與靜摩擦門檻。
- `circular/` — 等速圓周運動：向心加速度 a=v²/r，v⊥a，放手沿切線飛出。
- `shm/` — 簡諧運動三表徵：圓周投影＝彈簧振子＋單擺，正弦曲線與週期公式。
- `orbit/` — 行星軌道沙盒：萬有引力（velocity-Verlet），克卜勒一/二/三定律、等面積楔形。
- `vectors/` — 力的合成：可拖曳力向量，平行四邊形/頭尾相接，力平衡判定。
- `machines/` — 槓桿與滑輪：力矩平衡與滑輪組省力倍數，功守恆。
- `buoyancy/` — 浮力與密度：阿基米德原理，沒入比例=ρ物/ρ液，視重。

### 🟧 熱學館
- `kinetic/` — 分子運動箱：氣體動力論，實測壓力 vs 理論 NkT/V（誤差<0.6%），微觀導出 PV=nRT。
- `heating-curve/` — 比熱與加熱曲線：冰→水→蒸氣，熔化/沸騰潛熱平台與 Q=mcΔT。

### 🟩 波動館
- `standing-wave/` — 駐波弦：共振、諧波 n=1,2,3、節點/腹點，f_n=nv/2L。
- `doppler/` — 都卜勒效應：波前壓縮/拉伸、聽到頻率升降、超音速音爆。
- `sound/` — 聲波合成器：響度/音調/音色，拍音（Web Audio 真實發聲）。
- `superposition/` — 干涉與摩爾紋：同一種疊加（概念整合 hub）。

### 🟪 光學館
- `lens/` — 透鏡成像（凸/凹）：三主光線追跡，實/虛、正/倒、放/縮，1/f=1/p+1/q。
- `refraction/` — 折射與全反射：Snell 定律、臨界角、光纖原理。
- `color-mix/` — 加法混色盤：RGB 三光疊加（黃/青/洋紅/白），對照顏料減法。
- `diffraction/` — 繞射成像：雙/單狹縫干涉繞射，相位差與前後距離差。
- `moire/` — 摩爾紋（Moiré）：兩組光柵疊加。
- `spectral-lab/` — 光譜干涉實驗室（3D 三原色干涉光場，Plotly）。
- `shadowless-lamp/` — 外科無影燈（2D 概念版）；3D 專業版見 `shadowless-lamp-sim` repo。

### 🟥 電磁館
- `circuits/` — 電路串並聯：歐姆定律、串/並聯電流電壓分配與燈泡亮度。
- `efield/` — 電場與電位：點電荷電場向量、等位面、庫侖定律（可拖曳）。
- `rc-circuit/` — RC 充放電：指數曲線、時間常數 τ=RC、63%/37%。
- `lorentz/` — 磁場中的帶電粒子：勞侖茲力 F=qv×B、迴旋半徑 r=mv/qB。
- `induction/` — 電磁感應發電機：法拉第定律 EMF=-dΦ/dt、磁通與電動勢 90° 相位。

### ⬛ 近代物理館
- `photoelectric/` — 光電效應：臨界頻率、強度無關、KE=hf−W，量子物理開端。
- `half-life/` — 放射性半衰期：隨機衰變、N=N₀(½)^(t/T)、統計漲落。
- `bohr/` — 波耳原子模型：能階躍遷、氫原子光譜（Hα 656nm、巴耳末/來曼/帕申系）。

### 🛠 工程・資料
- `hovercraft/` — RC 氣墊船工程模擬器（React + Babel）。
- `housing/` — 全台房市實證數據系統（React + Chart.js）。

> 註：`hovercraft/`、`housing/`、`spectral-lab/` 透過 CDN 載入函式庫，需連網；力學/熱學/波動/光學/電磁/近代各館為純 HTML/Canvas，可離線。
