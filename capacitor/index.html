平行板電容器

兩片平行金屬板接上電壓 V，會在板上儲存等量異號電荷 Q，板間建立均勻電場 E。電容 C = ε₀A/d（板越大、越近電容越大）， 電荷 Q = CV、板間電場 E = V/d、儲存能量 U = ½CV²。插入介電質可讓電容變大 εr 倍。

情境：
物理重點：平行板電容 C = ε₀·εr·A/d（ε₀=8.85×10⁻¹² F/m）：板面積 A 越大、間距 d 越小，電容越大。 接上電壓後 Q = CV（電容決定同電壓下能存多少電荷）；板間是均勻電場 E = V/d；儲存的電能 U = ½CV² = ½QV = Q²/2C。 插入介電質（εr>1）會削弱板間電場、使電容變 εr 倍，能存更多電荷。電容器用於濾波、計時(配合電阻)、閃光燈儲能、觸控螢幕感測。
