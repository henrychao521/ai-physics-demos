電磁・RC 電路

RC 電路：充電與放電

電容透過電阻充電／放電，電壓與電流呈指數變化。時間常數 τ = RC 是快慢的量度： 充電經過一個 τ 充到約 63%，放電降到約 37%；約 5τ 後幾乎完成。R 或 C 越大，過程越慢。

電路（電容填充＝電量）
電容電壓 V_C（青）與 電流 I（黃）對時間
物理重點：充電時 V_C(t) = V(1 − e^(−t/τ))、電流 I(t) = (V/R)e^(−t/τ)（電流由大變小，電容充飽就不再流）。 放電時 V_C(t) = V₀·e^(−t/τ)。時間常數 τ = RC：t = τ 時充到 1−1/e ≈ 63.2%，t = 5τ 時 > 99%。 這個指數行為是計時電路、相機閃光燈充電、心律調節器、訊號濾波（高通／低通）的基礎。注意電流在開關瞬間最大、之後衰減。
