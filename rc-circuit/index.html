電磁・RC 電路

RC 電路：充電與放電

電容透過電阻充電／放電，電壓與電流呈指數變化。時間常數 τ = RC：充電一個 τ 充到約 63%、 放電降到約 37%，約 5τ 完成。R 或 C 越大過程越慢。

電路（電容填充＝電量）
電容電壓 V_C（青）與 電流 I（黃）
物理重點：充電 V_C(t) = V(1 − e^(−t/τ))、電流 I(t) = (V/R)e^(−t/τ)（電流由大變小）。放電 V_C(t) = V₀·e^(−t/τ)。 時間常數 τ = RC：t = τ 充到 1−1/e ≈ 63.2%，t = 5τ 時 > 99%。這是計時電路、相機閃光燈充電、訊號濾波的基礎。電流在開關瞬間最大、之後衰減。
