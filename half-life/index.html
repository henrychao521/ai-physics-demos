放射性半衰期

放射性原子何時衰變是隨機的，無法預測單一顆；但一大群原子整體上每經過一個半衰期 T，就有一半衰變掉。 所以數量按 N = N₀·(½)^(t/T) 指數衰減：T 後剩 1/2、2T 後剩 1/4…

情境：
原子（綠＝未衰變，灰＝已衰變）
剩餘數量 N(t)：實際(綠) vs 理論 N₀(½)^(t/T)(虛線)
物理重點：衰變是機率事件——每顆原子單位時間有固定機率衰變，與它「活了多久」無關（無記憶）。對大量原子， N(t) = N₀·(½)^(t/T) = N₀·e^(−λt)，衰變常數 λ = ln2 / T。碳-14（5730 年）定年、碘-131（8 天）醫療、鈾-238（45 億年）推定地球年齡。 原子少時曲線鋸齒不平滑（統計漲落），越多越貼近理論——這就是統計物理的精髓。
