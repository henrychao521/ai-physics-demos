電磁・電流磁效應

電流的磁效應（安培右手定則）

電流會在周圍產生磁場。直導線外磁場成同心圓 B = μ₀I/2πr（右手握住導線、拇指指電流、四指繞磁場）； 圓線圈中心 B = μ₀I/2R；螺線管內部接近均勻 B = μ₀nI，像條形磁鐵。拖曳探針量任一點磁場。

提示：
物理重點：電流生磁（厄斯特發現）。直導線：磁場繞導線成同心圓、B = μ₀I/(2πr)，離導線越遠越弱（∝1/r）。 圓線圈：把直導線彎成圈，中心磁場 B = μ₀I/(2R)。螺線管：許多圈疊起來，內部磁場近乎均勻 B = μ₀nI（n＝每公尺匝數），外形如條形磁鐵（一端 N、一端 S）。μ₀=4π×10⁻⁷ T·m/A。右手定則判方向：直導線握住、拇指順電流則四指繞向即磁場；線圈/螺線管四指順電流則拇指指向內部 N 極。
