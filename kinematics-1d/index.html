力學・運動學

一維等加速度運動（x-t・v-t・a-t）

調初速度 u 與加速度 a，看物體沿軌道運動，並對照三張圖：x-t（位置）、v-t（速度）、a-t（加速度）即時連動。 v = u + at（v-t 是直線，斜率＝a）、x = ut + ½at²（x-t 是拋物線，斜率＝v）。a<0 為減速，速度歸零即停止。

情境：
物理重點：等加速度運動三公式：v = u + at、x = ut + ½at²、v² = u² + 2as。 圖形意義：v-t 圖斜率 = 加速度 a、v-t 圖面積 = 位移；x-t 圖斜率 = 速度 v。 a=0 → 等速（x-t 直線）；a>0 → 加速（x-t 上凹拋物線）；a<0 → 減速，速度歸零後停止（如煞車）。自由落體是 a=g≈9.8 m/s² 的特例。
