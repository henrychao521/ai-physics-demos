## 段落 1：球面鏡成像（凹面鏡／凸面鏡）　mirror/　✅ 上線 commit 6dee030

### 設計決策
- 補光學館缺口（原本只有透鏡 lens，無球面鏡），課綱必考主題。
- 複用 lens 的版面 / 滑桿 / 公式面板 / 拖曳架構（最省力、與全站風格一致）。
- 關鍵物理差異 vs 透鏡：① 焦距 f=R/2（曲率半徑）；② 凹面鏡 f>0、凸面鏡 f<0；③ 光「反射」非「折射」；④ 實像在鏡前（與物同側）→ 像座標 x=−q（透鏡是 +q）。

### 物理（先以 node 驗證，再建檔）
鏡公式 1/f=1/p+1/q（同透鏡形式），q=p·f/(p−f)，放大率 m=−q/p，像高 h′=m·h。
node 驗證（R=30→f=±15）：
- 凹遠物 p=58 → q=20.23(實) m=−0.349(倒立縮小) ✓
- 凹 C~F p=22 → q=47.14 m=−2.143(實・倒立・放大) ✓
- 凹 F內 p=9  → q=−22.5(虛) m=2.5(正立放大＝刮鬍鏡) ✓
- 凹 在C p=30 → q=30 m=−1(實・倒立・等大) ✓
- 凸 p=35     → q=−10.5(虛) m=0.3(正立縮小＝後照鏡) ✓

### 實作要點
- 鏡面畫成弧線（凹向右凸 / 凸向左凸）+ 反射面高光 + 鏡背斜線。
- 反射光線追跡 reflRay()：實像→鏡前實線；虛像→實線往前 + 虛線延伸到鏡後虛像。兩條主光線（平行→反射過 F、射頂點→對主軸對稱反射）依建構通過已驗證的像點，幾何正確。
- 即時公式面板：f=R/2 → 1/f=1/p+1/q → q → m → h′ + 參數關係（f=R/2、凹 F內放大虛像、凸恆縮小虛像、p=f 時 q→∞ 探照燈）。
- 滑桿：物距 p、曲率半徑 R（教 f=R/2）、物高 h；情境：遠物 / C~F / 刮鬍鏡 / 後照鏡；可拖曳物體。

### 驗證（實機）
- 預設 p=50,R=30：desc「實像、倒立、縮小　q=21.4cm m=−0.43」；公式面板數值自洽；stats 一致。
- 三情境切換正確：凸後照鏡 q=−10.8(虛縮小)、刮鬍鏡 q=−22.5(虛放大)、C~F q=47.1(實放大)。
- 首頁光學館新增卡片（含縮圖 SVG），totalCards 34→35，手機無水平溢出，全程零 console 錯誤。
- 過程插曲：原預覽伺服器已結束、現存伺服器工作目錄不在專案 → 重建本機預覽設定（已 gitignore）並啟新伺服器後正常。

### 產出

## 段落 2：理想氣體 PV 圖　gas-law/　✅ 上線 commit 29ad917

### 設計決策
- 補熱學最大缺口（原僅氣體動力論、加熱曲線兩個）。
- 雙畫布：左＝活塞氣缸（分子運動具象化）、右＝PV 圖（抽象關係視覺化），兩者連動。
- 以「過當前點的三條導引線（等溫雙曲線／等壓水平／等容鉛直）」直接教三大過程；拖曳留軌跡讓過程形狀浮現。

### 物理
- PV=nRT（R=8.314 J/mol·K），P=nRT/V（單位 kPa，因 kPa·L=J）；分子均方根速率 v_rms ∝ √T。
- 三定律為其特例：波以耳（等溫 P∝1/V）、查理（等容 P∝T）、給呂薩克／查理（等壓 V∝T）。

### 實作要點
- 氣缸：N 顆粒子（∝n）在活塞左側彈跳，速率 ∝√(T/300)，活塞 x ∝V，撞活塞壓力箭頭（∝P）。
- PV 圖：參考等溫線（150/300/450/600K，當前 T 高亮）、白色狀態點、等壓(cyan 水平)/等容(green 鉛直)導引線、拖曳軌跡(虛線)。
- 即時公式面板：P=nRT/V→結果、PV=nRT、v_rms∝√T + 三定律關係。
- 過程示範鈕：等溫壓縮/等壓加熱/等容加熱，用 animateTo 緩動描繪曲線形狀。

### 驗證（實機）
- 預設 V8/T300/n1 → P=312kPa；slider/stats/計算三者一致。
- 等溫：V 8→4(減半) → P 312→624(加倍)、PV 守恆 2494J ✓（波以耳）。
- 等容：T 300→600(加倍) → P 624→1247(加倍) ✓（查理）。
- PV 圖渲染正確（等溫線/狀態點/導引線/軌跡），零 console 錯誤。
- 過程修正：原用「loop 函式包裝」hack 做動畫有 bug（V 不變）→ 改成把 stepAnim(dt) 直接併入主 loop（乾淨）。另：背景分頁 rAF 會暫停，headless 測試時動畫/粒子凍結屬正常，前景使用正常。

## 段落 3：斜面與力的分解　incline/　✅ 上線 commit 5e2fb43

### 設計決策
- 經典必考；目前 newton-force 只有水平面，補「斜面重力分解」。
- 視覺核心：把重力 mg 同時畫出「沿斜面 mg·sinθ」與「垂直 mg·cosθ」虛線分量，讓學生看見分解。

### 物理（node 驗證 4 情境）
- N=mg·cosθ；下滑力 mg·sinθ；最大靜摩擦 μN。門檻：mg·sinθ vs μN → 兩邊除 mg·cosθ → tanθ vs μ（與 m 無關）。滑動 a=g(sinθ−μcosθ)。
- node：θ25μ0.3→滑a=1.48、θ15μ0.3→靜止、θ45μ0.2→a=5.54、θ30μ0→a=4.9(=g·sinθ)。

### 實作要點
- 斜面三角形（右下為底端、坡面往左上）+角度弧；方塊沿斜面旋轉貼合；滑動時沿坡加速下滑動畫（到底重置）。
- 力箭頭從質心：mg(下,橙)、N(外法線,cyan)、f(上坡,rose) + 重力分量(虛線：沿坡下、壓向面)。
- HUD：狀態(靜止●/下滑▼)、N、f、a。公式面板：N→下滑力→μN→門檻判定→a + 關係。

### 驗證（實機）
- θ25/m2/μ0.3：HUD 下滑 N17.8 f5.3 a1.48；公式自洽，與 node 一致。
- 情境：無摩擦(冰) a=4.14=g·sin25、緩坡靜止 a=0(f=mg·sinθ 平衡)、陡坡大摩擦 a=2.55。零 console 錯誤。

## 段落 6：熱傳導與熱平衡　heat-transfer/　✅ 上線 commit 1c9c64d

### 設計決策
- 補熱學量熱（原 kinetic/heating-curve/gas-law 沒有「混合終溫」）。用兩方塊接觸 + 溫度-時間收斂圖直觀呈現熱平衡。

### 物理
- T_f=(m₁c₁T₁+m₂c₂T₂)/(m₁c₁+m₂c₂)；m₁c₁(T₁−T_f)=m₂c₂(T_f−T₂)；Q=mcΔT。
- 動畫用指數鬆弛 T_A(t)=T_f+(T₁−T_f)e^(−t/τ)、T_B 同 τ；數學上 m₁c₁T_A+m₂c₂T_B 任意 t 皆=定值（能量守恆）。

### 實作要點
- 兩方塊依溫度上色(藍冷→紅熱)、高度∝√m、熱流箭頭(熱→冷)、接觸面。溫度-時間圖兩曲線 + 終溫綠虛線 + 游標。
- 材質 水(4.18)/鋁(0.90)/鐵(0.45) 循環切換。公式面板：熱容量 mc / 終溫 / 交換熱量 + 關係（mc 大者主導）。

## 段落 9：核分裂連鎖反應　nuclear-fission/　✅ 上線 commit 8bb160c

### 設計決策
- Tier 2 核物理。用蒙地卡羅粒子模擬呈現「增殖係數 k」如何決定次/臨界/超臨界。

### 物理
- 第 n 代分裂 ∝ kⁿ；k<1 熄滅、k=1 穩定、k>1 爆增。每次分裂≈200MeV(質量虧損 E=Δmc²)；1g U-235≈8.2×10¹⁰J(≈2.5噸煤)。

### 實作要點

- 點畫面/按鈕射入中子；HUD 分裂數/飛行中子/剩餘/能量；閃光與中子拖尾。
