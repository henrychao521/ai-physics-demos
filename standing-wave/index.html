駐波弦：共振與諧波

兩端固定的弦被驅動振動。只有頻率剛好等於諧波頻率 f_n = n·v/2L 時，反射波與入射波疊加成駐波、 振幅變大（共振），出現固定的節點（不動）與腹點（振幅最大）。掃動頻率找共振。

快速跳到諧波：
物理重點：弦上波速 v = √(張力 T / 線密度 μ)。兩端固定 → 弦長必是半波長整數倍 L = n·λ/2， 得諧波頻率 f_n = n·v/2L。驅動頻率落在 f_n 才共振、振幅放大；偏離就互相抵消、幾乎不動。 張力越大波速越快、共振頻率越高；弦越長頻率越低——這就是弦樂器調音原理。第 n 諧波有 n 個腹點、n+1 個節點。
