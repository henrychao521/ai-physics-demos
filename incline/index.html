力學・牛頓運動定律

斜面與力的分解

斜面上的物體，把重力 mg 分解成沿斜面 mg·sinθ（使其下滑）與垂直斜面 mg·cosθ（壓向表面）。 正向力 N = mg·cosθ，最大靜摩擦 μN。當 mg·sinθ ≤ μN（tanθ ≤ μ）靜止；超過則下滑，加速度 a = g(sinθ − μcosθ)。

情境：
物理重點：選沿斜面與垂直斜面為座標軸，重力 mg 分解為 mg·sinθ（沿面）與 mg·cosθ（垂直）。 垂直方向平衡：N = mg·cosθ。沿面方向：下滑力 mg·sinθ vs 最大靜摩擦 μ_s·N = μ_s·mg·cosθ。 兩邊同除 mg·cosθ 得門檻 tanθ vs μ_s——與質量無關！滑動後 a = g(sinθ − μ_k·cosθ)。角度越陡、摩擦越小，加速度越大。
