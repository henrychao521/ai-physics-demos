時間膨脹（光鐘）

光速對每個人都一樣。靜止鐘的光子直上直下，一個來回就「滴答」一次；運動鐘的光子得走更長的斜路徑， 但光速不變，所以一次滴答要更久——這就是運動的鐘走得慢。膨脹倍率即勞侖茲因子 γ = 1/√(1−β²)，Δt = γ·Δt₀。

速度：
物理重點：用畢氏定理就能導出時間膨脹。光鐘鏡距 D，靜止時一次滴答 Δt₀ = 2D/c。 當鐘以速度 v 運動，光子走斜邊：(c·Δt)² = (2D)² + (v·Δt)² → Δt = γ·Δt₀，其中 γ = 1/√(1−v²/c²)。所以觀察者看「運動鐘」每滴答一次，自己已過了 γ 倍的時間。同時尺也會在運動方向收縮：L = L₀/γ。 β=0.5→γ≈1.15、0.9→2.29、0.99→7.09。日常 v≪c 時 γ≈1 幾乎沒差；但 GPS 衛星、加速器裡的粒子都必須修正。
